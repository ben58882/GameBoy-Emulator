Gameboy Emulator

A work in progress frame accurate ( ~60 fps) gameboy emulator written in Java.

Features:

CPU
An accurate implementation of gameboy's SHARP LR35902 cpu. Passess all of Blarg's rom tests.

PPU
Pixel accurate rendering of sprites, background and window.

ROMS
Supports stand alone catridges, mbc1, mbc2 and mbc3 roms with timer and saved files support.

Features I hope to impelment:
APU for audio
mbc5, mbc6 and mbc7 support

To use:
1) download a gameboy rom (.gb), some gameboycolor roms are also supported (.gbc).
2) paste the file path where indicated in the Main.java file.
3) compile the files by running javac *.java in the terminal
4) launch the program by running java Main in the terminal

Saved files support:
For supported games a .sav file will be generated within the folder upon launching the program, similarly if there is a .sav file with the same name as the rom in the folder the emulator 
will fill in the internal ram with the .sav contents. Note that .sav files generated by this emulator are unlikely to be compatible with .sav files from other emulators. Only games
that had a save functionality on the original gameboy (game catridges that came with a battery + ram) will generate and read .sav files.

Button layouts:
A button mapped to A key on keyboard
B button mapped to S key on keyboard
START button mapped to ENTER key on keyboard
SELECT button mapped to LEFT SHIFT key on keyboard
ARROW buttons mapped to ARROW keys.
